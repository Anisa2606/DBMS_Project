# FreelanceHub DBMS Project

 Project Title
FreelanceHub: Decentralized Gig Matching and Payment Escrow System

---

Description
FreelanceHub is a database-driven system designed to manage freelance activities efficiently. It integrates project posting, bidding, contract management, milestone tracking, and escrow-based payment processing into a structured relational database system.

The project demonstrates core Database Management System (DBMS) concepts such as normalization, constraints, relationships, triggers, stored procedures, and functions.

---

## 🎯 Objectives
- To design a relational database for freelance project management
- To ensure secure and transparent payment handling using escrow
- To automate processes using triggers and stored procedures
- To maintain data integrity using constraints and normalization

---

 Technologies Used
- MySQL
- SQL & PL/SQL
- PHP
- HTML
- CSS

---

Database Features
- Primary and Foreign Key constraints
- Normalized tables (up to 3NF)
- Trigger for automatic payment release
- Stored Procedure for contract creation
- Function to check escrow balance
- Transaction handling
